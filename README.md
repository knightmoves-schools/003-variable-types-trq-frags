# 003 Variable Types

[![003 Variable Types](https://img.youtube.com/vi/P69ekJ981Iw/0.jpg)](https://www.youtube.com/watch?v=P69ekJ981Iw)

the VariableTypes.cs file
- should define a `public` class-level string variable named `AnimalName`
- should define a `public` class-level character variable named `GradeLetter` (like: A, B, C, D, or F)
- should define a `public` class-level numeric whole number variable named `Year`
- should define a `public` class-level numeric decimal number variable named `Height` (like: 5.5, 6.2, etc)
- should define a `public` class-level boolean variable named `IsIndoorPet`
